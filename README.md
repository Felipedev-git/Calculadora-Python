# Calculadora Python

Este projeto foi desenvolvido durante meus estudos de Lógica de Programação e Python. O objetivo foi criar uma calculadora funcional via terminal, aplicando conceitos progressivos de desenvolvimento.

## Evolução do Projeto

* **Versão 1 (`CALCULADORA_v1.py`):** Lógica estruturada utilizando loops `while`, condicional `if/elif/else` e tratamento de erros básicos.
* **Versão 2 (`CALCULADORA_v2.py`):** Refatoração do código utilizando **Funções (`def`)** para modularizar as operações matemáticas, tornando o código mais limpo e escalável.

## Funcionalidades

- [x] Operações básicas: Soma, Subtração, Multiplicação e Divisão.
- [x] Loop infinito (o programa não fecha após cada conta).
- [x] **Tratamento de Erros:** Proteção contra inputs inválidos (ex: digitar letras) usando `try/except`.
- [x] Opção de saída controlada pelo usuário.

## Como rodar

Certifique-se de ter o Python instalado.

```bash
git clone [https://github.com/Felipedev-git/Calculadora-Python.git](https://github.com/Felipedev-git/Calculadora-Python.git)
cd Calculadora-Python
python CALCULADORA_v2.py
